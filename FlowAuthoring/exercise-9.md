# Exercise 9

Everybody is super exited by your newly created application! Since you plan on releasing more applications you figured it is a good idea to create a Gallery App.

**Assignment**:

Create a gallery app and add your application from exercise 8. Add a second application. You can for instance create a second application from one of the Sample workspaces. Also add a link to a website. For instance, [http://safe.com](http://safe.com).

<details>

<summary>Tips:</summary>

* You can create the layout of a gallery app on the same way you did for the pervious app.
* Make sure to write information about the gallery app.
* You can add tooltips for each app.
* Test the gallery app.

</details>
